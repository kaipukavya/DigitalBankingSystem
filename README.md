# Digital Banking System

## Overview
Digital Banking System is a Java-based banking application developed using **Core Java, OOP concepts, JDBC, and Oracle Database**. It provides basic banking operations like customer management, account handling, deposits, withdrawals, and balance checking.

## Features
- Add Customer
- View Customer Details
- Update Customer Profile
- Delete Account
- Deposit Money
- Withdraw Money
- Check Account Balance

## Technologies Used
- Java (Core Java, OOP)
- JDBC
- Oracle Database
- Eclipse IDE

## Concepts Implemented
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Database Connectivity using JDBC

## How to Run
1. Clone the repository.
2. Configure Oracle Database connection.
3. Add Oracle JDBC Driver.
4. Run `Main.java`.
